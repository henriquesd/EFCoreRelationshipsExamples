# EFCoreRelationshipsExamples
Console application with examples of relationships using EF Core 3.1 and Fluent API

## Technologies
- .NET Core 3.1
- Entity Framework 3.1
- Fluent API

## Examples of Relationships
- One-to-one (1:1)
- One-to-Many (1:N)
  - Fully Defined Relationships
- Many-to-Many (N:N)
